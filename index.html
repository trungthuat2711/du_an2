Steps to deploy website using VScode

  1. Tạo repository trên github: dùng để lưu trữ dự án trên máy chủ git.
  2. Tạo dự án trên máy tính cá nhân: dùng để code và lưu trữ cục bộ trên máy tính.(ổ đĩa hỏng -> mất dữ liệu).
  3. Khởi tạo git: để tạo 1 kho lưu trữ - Local repo
    • 3.1: Mở terminal
    • 3.2: config mặc định chọn git bash khi mở terminal
    • 3.3: kiểm tra trạng thái git bằng lệnh ( git status )
    • 3.4: Để khởi tạo 1 kho lưu trữ git mới trên local ( git init ) sau bước này đã có thư mục git để sẵn sàng lưu trữ thay đổi, nhưng chưa có gì được theo dõi
  4. Add thư mục dự án vào staging area chờ commit:
    • 4.1: Add tất cả: ( git add . )
    • 4.2: Add lẻ thư mục, tệp: ( git add ten_thu_muc )
  5. Commit và đẩy code lên github: ( git commit -m "first commit" )
    • 5.1: git config --global user.email "ntt27112005@gmail.com"
    • 5.2: git config --global user.name "thuat"
  6. git branch -M main: chuyển tên nhánh master sang main
  7. git remote add origin https://github.com/trungthuat2711/du_an2.git
  8. git push -u origin main
